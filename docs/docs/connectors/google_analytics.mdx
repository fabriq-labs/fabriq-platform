# Google Analytics

To set up the Google Analytics source connector, please follow the instructions outlined below:

#### Configuration:

- Enter your Service Account Key JSON.
- Provide your [view ID](https://developers.google.com/analytics/devguides/reporting/data/v1/property-id?_gl=1*cctu2a*_ga*MTAyMzEzNzY4OS4xNjk2ODM5NDkw*_ga_EDX3TPP6H7*MTY5NzAyODQ0OC4xMy4xLjE2OTcwMjg0NTguMC4wLjA.#what_is_my_property_id).
- Specify the Start Date from which to replicate report data in the format **YYYY-MM-DD**.
- Add Custom Reports as a JSON array describing the custom reports you want to sync from Google Analytics.

Once you have entered the required details, update them in the Fabriq UI, and click the **Save** button.
